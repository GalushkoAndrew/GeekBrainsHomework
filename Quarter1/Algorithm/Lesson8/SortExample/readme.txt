Написал корзинную сортировку (Bucketsort) для сортировки массива целых чисел.
Перед собственно сортировкой добавил алгоритм распределения по корзинам
Заложил ограничения, что в корзину помещаем не менее 200 значений и будет выделяться не более 5 корзин.
Возможно, есть лишние предварительные вычисления, но ими старался сократить дальнейшие операции по сортировке.

Для сортировки выбрал быстрый алгоритм (сортировка Хоарда)

+ Порисовал на бумажке на примере случайного массива пробежался по алгоритму сортировки вручную, чтобы закрепить его как собственный опыт.

Тестов по скорости не делал, ни с чем не сравнивал и задачу со звездочкой решил не решать.
Вообще эта корзинная сортировка странная.
